### 📖 Sobre o Projeto
Este comparador de preços foi desenvolvido durante as aulas de Arquitetura de Software. O foco principal foi a aplicação de conceitos fundamentais de desenvolvimento web e manipulação de dados em tempo real.

### 🛠️ Tecnologias & Conceitos
* **Frontend:** HTML5, CSS3 (Flexbox/Grid) e JavaScript Vanilla.
* **Consumo de API:** Integração com [Nome da API se houver] para busca de produtos.
* **DOM Manipulation:** Atualização dinâmica da interface sem recarregamento de página.

### 💡 O que eu aprendi
Neste projeto, foquei em entender como estruturar uma aplicação que lida com múltiplos fluxos de dados, garantindo que o usuário tenha uma resposta visual rápida ao buscar um produto.
