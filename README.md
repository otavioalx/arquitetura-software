# 🔍 Price Comparator - Academic Lab


### 📖 Sobre o Projeto
Este comparador de preços está sendo desenvolvido durante as aulas de Arquitetura de Software. O foco principal foi a aplicação de conceitos fundamentais de desenvolvimento web e manipulação de dados em tempo real. Neste projeto, foquei em entender como estruturar uma aplicação que lida com múltiplos fluxos de dados, garantindo que o usuário tenha uma resposta visual rápida ao buscar um produto.
